# My Personal Site
Personal Site rebuilt using Astro rather than vanilla HTML and JS. This time more preformat and accessible.

**Features:**
- About me page
- Index of my blogs
- A Bucket list of things I want to learn/ try
- A Gallery of my Illustrations
- A Link to my FAQ Page

## Tech Stack
- [Astro JS](https://astro.build/)
- SCSS
- [VueJs](https://v3.vuejs.org/)
